🇵🇭 Philippines BIR

This extension helps Philippine businesses generate BIR-compliant tax returns and reports using data already in Manager.io — no double-entry needed.

Reports

Install BIR report buttons into any business. Once installed, each report appears under that business's Reports tab. Reports marked Coming soon are planned for a future phase.

Tax codes

Create standard PH tax codes with one click. Also contains mapping sections for VAT, EWT / CWT, FWT, and Percentage Tax — configure once and all reports read from them automatically.

Business / Customers / Suppliers / Employees

BIR fields (TIN, RDO Code, classification, names, addresses, SSS / PhilHealth / Pag-IBIG numbers) stored directly in Manager records. Fill once; all reports use them.

Payslip items

Assign payslip items to BIR reporting categories so values flow into 1601C, SAWT, and 2316 correctly.
